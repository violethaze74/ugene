Major changes in UGENE 1.26

  • General:
    • Support of high-resolution Retina displays
    • By default, all documents are opened in tabs instead of windows. To change this parameter go to the Application Settings
  • NGS: All databases, supported by SnpEff, are now available for prediction of variant effects
  • Bug fixes and minor improvements

A new view for working with Sanger reads - the Chromatogram Alignment Editor - is planned for 1.27 version. Stay tuned!

Изменения в UGENE 1.26

  • Общие:
    • Поддержка мониторов с высоким разрешением Retina
    • По умолчанию все документы открыты в отдельных вкладках, а не в окнах. Изменить способ отображения можно в настройках приложения
  • NGS: Поддержка всех доступных баз данных SnpEff для предсказания эффектов вариаций
  • Исправление ошибок, мелкие улучшения

Выпуск нового редактора выравнивания хроматограмм, предназначенного для визуализации и анализа данных секвенирования по Сэнгеру, запланирован на версию 1.27. Следите за новосяти UGENE!
